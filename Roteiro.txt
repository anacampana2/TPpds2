Personagens:


O objetivo da POO é aproximar a programação cada vez mais próxima à realidade, por isso, num RPG
os personagens devem realizar funções básicas do cotidiano tais como:

// -Ver ao redor;
-Falar com;
//-Andar;
//-Correr;
//-Pular;
-Atacar:
    -Ataque corpo a corpo;
    -Ataque à distancia;
    -Magia;
//-Defender;

Além disso, existem varios tipos de personagens, os jogáveis e os não jogáveis (NPC), entre os não jogáveis temos:

//    -Habitantes comuns:
//        >Realeza;
//      >Mercadores;
//      >Donos de tabernas;
//        >Pessoas aleatórias na rua;

    -Monstros:
        >Minions;
        >Bosses;
        >Reis demonios;

Para cada tipo de personagem, temos classes e sexos diferentes (Podemos ou não excluir os tipos diferentes de raças, para dar menos trabalho no jogo). O jogador pode, ou não, escolher a classe do seu personagem.
